translate(self, scheme)

 
Translate an URL to a new scheme.
Parameters:
  • scheme (string) - the new scheme to translate into
Returns: URL
URL representation of the translated string
Raises:
  • NotImplemented -
  • BadParameter -
  • NoSuccess -
Notes:
  • if the scheme is not supported, a 'BadParameter' exception is raised.
  • if the scheme is supported, but the url cannot be translated to the scheme, a 'NoSuccess' exception is raised.
  • if the url can be translated, but cannot be handled with the new scheme anymore, no exception is raised. That can only be detected if the returned string is again used in a URL contructor, or with set_string().
  • the call does not change the URL represented by the class instance itself, but the translation is only reflected by the returned URL object.
  • the notes from section 'The URL Problem' of the GFD-R-P.90 document apply.
